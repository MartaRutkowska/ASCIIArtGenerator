
# ASCII art generator

Simple console app that let's you generate an ASCII art from chosen image.






## Run

To run, supply two arguments:\
{filePath} {options}

options:\
"-bw" for black and white image\
"-c" for colorful image
## Tech Stack

**Console app:** .NET Core 8
